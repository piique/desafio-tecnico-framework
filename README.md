# Desafio Técnico Framework

Desafio realizado conforme solicitado por email.

Link para vizualização do projeto: [https://desafio-framework.pedrovilaca.com/](https://desafio-framework.pedrovilaca.com/)

## Tecnologias e ferramentas utilizadas
- React;
- Axios (controle de requisições http);
- node-sass (scss);
- React Router Dom (manipulação de rotas);
- ESLINT (padronização de código);

## Requisitos para rodar o projeto
- Node.js - versão 14
- Git (opcional)

Obs: Para rodar o projeto em uma versão diferente da 14 do Node.js talvez seja necessária atualização da versão da dependencia node-sass no arquivo package

## Comandos para execução do projeto localmente
```
$ git clone https://github.com/piique/desafio-tecnico-framework.git
```
```
$ cd desafio-tecnico-framework/
```
```
$ npm install
ou
$ yarn
```
```
$ npm start
```
